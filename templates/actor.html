Classes: Actor

Actor (internally AActor) is the base class for all ZScript and DECORATE actors. When a parent class isn't declared in an actor definition, it uses Actor as the parent.

Actor contains and defines all the code to manage map objects such as decorations, monsters, powerups, players, and many more. It itself is derived from Thinker, which is the parent class for most classes that perform some kind of actions in the game's world — for example, a door or a crushing ceiling is a Thinker too, but not an Actor. Thinker itself is derived from Object, which is the base class for all classes and one of the main data structures in ZScript, alongside Structs.
